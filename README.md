# Web Development From Scratch

## Overview
This repository is a **beginner-friendly, spoon-feeding tutorial** designed for students who are learning web development fundamentals.

Each lesson focuses on **step-by-step commands** you can copy/paste and run, with **tiny explanations** that clarify *what you’re doing* and *why it matters*.

If you’re new to the terminal, Git, or GitHub, start with the first lesson below.

## Start Here
- **Lesson 01 (Git/GitHub):** [lesson-01-git/GIT-GITHUB-TUTORIAL.md](./lesson-01-git/GIT-GITHUB-TUTORIAL.md)

## How to Use This Repo
1. Open the tutorial link above.
2. Follow the steps in order (don’t skip).
3. Run the commands exactly as written.
4. If something breaks, re-read the small explanation next to the step—it usually explains what to check.
